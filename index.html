Paulo Alves

Sobre

Natural de Recife-PE é fã de tecnologia em geral, graduado em Análise e Desenvolvimento de Sistemas pela Universidade Estácio de Sá em 2016.

Já atuou no mercado com desenvolvimento de aplicações em nuvem usando o CRM Salesforce, possui certificação Scrum Fundamentals Certified, conhecimentos em algumas linguagens de programação e banco de dados.

Atualmente procura estar atualizado, sempre buscando novos desafios, fazendo cursos, debatendo ideias, procurando ajudar outros compartilhando conhecimento.

Habilidades

Desenvolvimento web - HTML5, CSS3, Javascript, Bootstrap, AngularJS, NodeJS, ASP.NET MVC5.

Linguagens Java, C#, Apex(Salesforce).

Banco de dados MySQL, SQL Server, MongoDB.

CRM Salesforce.

Metodologia Ágil Scrum.

Sistemas de controle de versão Git/GitHub.

Projetos

Projeto Produtos Marcas: aplicação desktop Windows desenvolvida na linguagem C# com Windows Forms, com a função de catalogar listas de produtos e marcas.

Projeto Tremanaweb Pub: página web para divulgação de um pub fictício.

Walking The Board: aplicação web que simula um jogo de obstáculos em um tabuleiro.

Repository Search: aplicação web responsável por fazer pesquisa de repositórios no GitHub.

Projeto Ciclo de Pagamentos: aplicação web utilizando o conjunto de tecnologias MEAN (MongoDB, Express, AngularJS e NodeJS) que simula um gerenciador para ciclos de pagamentos.
